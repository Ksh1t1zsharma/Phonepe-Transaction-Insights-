# Contributing

Thank you for your interest in contributing to PhonePe Transaction Insights! Please follow the guidelines below.

1. Fork the repository and create a branch for your change.
2. Keep changes small and focused; one feature / fix per PR.
3. Write clear commit messages and describe your changes in the PR description.
4. Update or add tests/documentation where appropriate.

For questions, open an issue on the original repository.
